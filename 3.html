JavaScript

JavaScript - JS for short - is a programming
language that is dynamically typed, prototype based (even for its object-oriented features), supports functional programming,
and is - in most cases - just-in-time compiled. Its syntax is similar to that of C or Java, but its concepts and runtime behavior
are distinctly different. The programming language JavaScript conforms to the international standard ECMAScript
